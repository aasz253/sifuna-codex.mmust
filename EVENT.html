Intro to Machine Learning Workshop

Date: December 5, 2025

Location: Virtual

Time: 2:00 PM - 5:00 PM (EST)

About the Workshop

This beginner-friendly workshop will introduce you to the exciting world of machine learning. No prior experience required! We'll cover:

  • Basic concepts of machine learning
  • Setting up your Python environment
  • Working with scikit-learn
  • Training your first model
  • Evaluating model performance

Requirements

Participants should have:

  • Basic understanding of Python (variables, loops, functions)
  • Laptop with Python 3.7+ installed
  • Jupyter Notebook (we'll provide installation instructions)
